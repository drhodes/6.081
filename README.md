I'm not affiliated with this course, but here is a quick way to going
with Xv6 on RISC-V, 6.081, found at the following URL

 https://pdos.csail.mit.edu/6.828/2019/


```bash

$ vagrant --version
Vagrant 2.2.6

$ VBoxManage -version
6.0.14r133895

$ vagrant up

... grab a cup of coffee ...

$ vagrant ssh

```
